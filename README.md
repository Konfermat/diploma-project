# diploma-project
Репозиторий для дипломной работы.

### Функционал сайта
* **Курсы:** Создание учебного материала и отображение его на сайте. Проверка знаний тестами.
* **Редактор курсов:** Создание, редактирование, удаление и т.д.
* **Каталог и навигация:** Просмотр курсов по темам и категориям.
* **Карта прогресса:** Отслеживание и планирование прогресса с помощью карты.
* **Обратная связь:**  Лайки, коментарии, оценка по пятибальной шкале. Рейтинг оценок, курсов, профилей.
* **Личный кабинет:**  Статистика, редактирование профиля, отслеживание обратной связи.
