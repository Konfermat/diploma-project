# чтобы pygraphviz установился сделай это
    # 1. Установка Graphviz
        pygraphviz — это лишь оболочка (wrapper) для основной программы Graphviz, 
        поэтому она не будет работать без нее.
        Скачайте EXE-инсталлятор (например, 64-bit EXE installer) с официального сайта Graphviz.
        Важно: Во время установки выберите опцию «Add Graphviz to the system PATH for all users» (или «for current user»). 
        Это позволит системе видеть исполняемые файлы программы.
    # 2. установи Visual Studio Build Tools !!! НЕ IDE !!!
        В официальном установщике (Visual Studio Installer) 
        выбери «Разработка классических приложений на C++»
    # 3. Укажи пути к файлам в powershell следующим образом
        pip install --config-settings="--global-option=build_ext" `
            --config-settings="--global-option=-IC:\Program Files\Graphviz\include" `
            --config-settings="--global-option=-LC:\Program Files\Graphviz\lib" `
            pygraphviz
